## 📌 Descripción

Este repositorio contiene distintos programas desarrollados en Python como parte de la materia de Fundamentos de Programación.

Los programas incluyen operaciones con matrices, cálculos de velocidad y otros ejercicios prácticos.

---

## 📂 Contenido

* `Matrices.py` → Operaciones con matrices
* `velocidad.py` → Cálculo de velocidad
* Otros archivos de práctica

---

## ⚙️ Tecnologías utilizadas

* Python 3
* Librerías básicas de Python

---

## 🚀 Cómo ejecutar los programas

1. Descargar el repositorio:

```
git clone https://github.com/kenia-hub12/mi_evidencia.git
```

2. Entrar a la carpeta:

```
cd mi_evidencia
```

3. Ejecutar un archivo:

```
python3 Matrices.py
```
